! AI and Art !


This is a collection of images generated using different emergent AI programs on the platform RunwayML. The first image present is a still drawn of Snow White and the Seven Dwarves in a program called SPADECOCO. This program takes user-drawn shapes in colors which have related image meanings and attempts to fit the image to each shape. The small collection underneath the first image was generated using a program called AdaIN which takes two images to synthesize the style of one with the image composition and subjects of the other. In this case classic representations of the death reaper were stylized with the colors and patterns of clown images
